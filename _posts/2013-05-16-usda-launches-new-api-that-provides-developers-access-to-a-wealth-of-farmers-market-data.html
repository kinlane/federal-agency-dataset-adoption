The United States Department of Agriculture (USDA) has just launched the brand new USDA Farmers Market Directory API providing developers programmatic access to the USDA National Farmers Market Directory database.According to the USDA website, the National Farmers Market Directory has over 7,800 farmers market listings for all 50 states and provides consumers farmers market information such as market locations, directions, operating times, product offerings, and accepted forms of payment.The USDA Farmers Market Directory API makes it possible for developers to access market directory data using search and data retrieval functionality.The data can then be used in websites and third-party applications.The API is REST based and can be accessed by using C#, Visual Basic, or AJAX JavaScript.
